# RESTer

RESTer is a library for making HTTP requests from a text file or string.

## Tests

To run tests:

```bash
npm test
```

Running tests will generate a code coverage report. To view:

```bash
npm test
open coverage/lcov-report/index.html
```

## Documentation

To generate documentation and view:

```bash
grunt jsdoc
open doc/index.html
```
